Supported Browsers

This knowledge base works best when displayed using the following browsers:

  • Google Chrome version 45 and higher
  • Internet Explorer version 11 and higher
  • Microsoft Edge version 13 and higher
  • Mozilla Firefox version 45 and higher

Google Chrome is the preferred browser and provides the best display.

Warning:  Internet Explorer versions 9 and lower are not supported!

Searching Text

You can use the search bar to find information quickly.

To view the search bar, click the magnifying glass icon in the toolbar.

To limit your search to a specific functional area of the knowledge base, select a category from the search filter. To do so, click the search filter icon and then from the drop-down menu select the functional area in which you want to search. For example, this selection will limit the search to only the Manufacturing (MFG) section:

While the search is restricted to one of the functional areas, the filter icon is displayed in black . To return to searching the entire knowledge base, click the black filter icon and select “All Files” from the drop-down menu. The filter icon turns white again.

Here are some examples of the syntax you can use for searching:

Type

To find

"find me"

(in quotes) Topics that include the exact phrase, “find me”.

find me

find+me

Topics that include both the words “find” and “me”, but not necessarily as a phrase.

find or me

find | me

Topics that contain either “find” or “me” (or both), in any order.

not “find me”

! ”find me”

Topics that do not include the phrase “find me”.

find ^ me

Topics that include the word “find” but not the word “me”.

Note:  If your search string includes a % symbol, the search does not return any results.
