Access Control Lists (ACL)

Access Control Lists (ACLs) are used to match traffic for processing by a specific overlay. Create ACLs from the Templates page, then push them to the appliances. You'll then be able to select from among them in the Traffic Access Policy on the Business Intent Overlays (BIO) page.

If using ACLs, we recommend the following:

  • Create an ACLAccess Control List. for realtime traffic to map onto the realtime overlay.
  • Create a default ACLAccess Control List. to catch all other traffic.
To Create an Access List
  1. From the Configuration tab, choose Access Lists.

  2. From the top of the window, click Manage Access Lists with templates.

    The Templates page opens showing the Access Lists form.

  3. Click Add ACLAccess Control List..

    The Create ACLAccess Control List. form opens.

  4. Enter a name, such as Default_ACLAccess Control List., then click Add.

    The form closes.

  5. Click Add Rule.

    A new line appears on the page.

  6. Make any changes that you need.

  7. Click to check the template check box, then click Save at the bottom of the Templates list.

    Your rule now matches your values. To keep it simple, you can save without changing any default values.

  8. Repeat this process to create a realtime overlay.
