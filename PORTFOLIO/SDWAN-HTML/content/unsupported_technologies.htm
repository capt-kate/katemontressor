Unsupported Technologies

These technologies are NOT currently supported:

  • Mixing 1Gbps and 10Gbps interfaces when using bridge or router modeOut-of-path deployment, where data traffic is redirected by using policy-based routing (PBR), Web Cache Coordination Protocol (WCCP), or Virtual Router Redundancy Protocol (VRRP)..
  • Flow redirection on 10Gbps interfaces.
  • 10Gbps Bonding.
  • Sub-Interfaces when the NAT option is enabled.
  • Communicating routing information with third party devices using routing protocols such as OSPF or BGP.
  • Terminating an MPLSMultiProtocol Label Switching is an IETF initiative that integrates Layer 2 information into Layer 3 (IP) packets. link on the Silver Peak.
  • Local InternetA global network of interconnected computers, enabling users to share information along multiple channels. access when WANWide Area Network hardening is used.
  • Multicast traffic is dropped while using In-Line Router ModeOut-of-path deployment, where data traffic is redirected by using policy-based routing (PBR), Web Cache Coordination Protocol (WCCP), or Virtual Router Redundancy Protocol (VRRP). when reliant routing protocols are not able to communicate across the Silver Peak.
