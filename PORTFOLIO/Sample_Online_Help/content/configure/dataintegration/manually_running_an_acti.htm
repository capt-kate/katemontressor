Manually Running an Activated Connection

Once a connection has been activated, it runs at the scheduled time. However, you can manually run the import connection at any time.

1. From the Integration workspace, click Start. The list of configured connections appear. If the connection is active, run now appears in the Mapping column.
2. Click Run Now. The confirmation window appears. Click Yes.
3. If your connection was defined as manual (see Data Import Configuration), you must upload a new data file in the format you specified (such as CSV, XSL, or XSLX). Click Upload.

The data is imported into the database.
