Run Program History

The Run Program/Export History page lists each time the run program export action has been run and the result. It is a convenient way for you to determine if your exports are working.

1. CFrom the Configuration Console Settings pane, click Monitor > Application Logs > Run Program History. The Run Program / Export History page appears.

The Log Level column shows the success or failure of the run program action.

2. Double-click an item from the list to view the audit history.
