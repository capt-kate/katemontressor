Styles

You can change the way HEAT looks to your users, including colors, fonts, and logos. All styles comply with standard CSS rules and can be defined with or without expressions (see Using Expressions in Styles).

There are two ways to use styles in HEAT:

  • Style Editor: You can define the look and colors of the overall application, including logo, headers, footers, panels, and workspaces. See Using the Style Editor.
  • Highlighter Styles: (formerly known as the Grid Highlighter). You can create your own styles or use the default styles to be applied to grids, forms (see Using Forms), and controls. See Using Highlighter Styles.
