* * * New or Improved for 2014.1 * * *  Employment Type

The Employment Type workspace enables you to define various types of employees.

The default types are:

  • Permanent
  • Contractor
  • Contractor - Part Time
Create an Employment Type
1. From the Configuration Console, click Configure > Organization Profile > Employment Types. The Employment Type workspace appears.
2. Click New Employment Type. A blank field appears.
3. Enter a unique name for your employment type into the field.
4. Click Save from the toolbar.

The employment type now appears on the list.
