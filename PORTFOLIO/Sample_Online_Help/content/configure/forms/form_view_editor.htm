* * * New or Improved for 2013.2 * * *  The New Form View Editor

Users will be able to access our new Form Editor Within the Configuration console. This new editor provides an easier, more customizable interface for creating forms.

Access the new Form View Editor
1. Within the Configuration console, open a Business Object and select the Layout tab.
2. Select a layout from the list, or click Add New Layout. The Layout page appears.
3. Click Add New Form View. The Form View Editor appears.

Classic Form View Editor

4. Click Try our new look! The new Form View Editor appears.

New Form View Editor

Using the new Form View Editor

The Form View Editor consists of the following:

  • Toolbox - contains elements to use in the form.
  • Toolbar - contains the buttons that appear in the toolbar area of the form. For more information on this feature, see The Toolbar Editor.
  • Layout - contains the main part of the form, including any tabs.
1. From the Toolbox, expand one of the panels to find the element you want to use.
2. Drag the element from the Toolbox onto the Layout. The areas where you can place the element become highlighted.
  • Dragging it onto the top half of the Layout places the element in the top half of the form.
  • Dragging it onto Add New Tab places the element in a tab, and the FormView Child Panel Relationship Editor appears.

    The Tab name is automatically created.

3. Click Edit Properties to set the form properties.

Form properties

-properties-
Property Description
Display Name Enter a unique name that appears to users.
Enable Object View Auditing Check to automatically show audit information.
Enable smart fill on repetitive input of some values Check to enable smart fill. After three iterations of selecting the same value, users will see the same value selected from the fourth time on.
Enable Next/Previous buttons on the search bar Check to enable the Next and Previous buttons on the toolbar. This enables the user to move from one record to the next.
Enable Cost Management Check to enable costing for this form. Click Edit Cost Settings to set costing information. For more information, see .
Enable Matching panel Check to create a panel that enables the user to access Object Matching.
After saving object show Optional. Select the form to show after the user saves the object. Otherwise, the original form remains.
4. Click Save from the toolbar.
  • To remove an item from the layout, click the header, then click Delete.

Form Delete button

  • To modify the element, click the header, then click Edit.
