Address Book

Enables the user to access the Address BookA HEAT object that stores names and email addresses from the employee and external contact objects. Users simply start typing a name in an address-book-enabled field, and a list of names and email addresses appears.. This control only applies to text fields.

Shows incident response time in relation to target response time. The states are:

  • blank: The incident record has been created recently. Response window still open.
  • met: Response was on time.
  • breached: No response, or late.
