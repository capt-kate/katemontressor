Viewing the LDAP Import Log

The LDAP Import log is included by default and lists all imports executed through LDAP.

Viewing the Workspace
1. Within the Configuration Console, open the LDAP Import Log workspace. The list of import jobs appears.
2. If you do not see any recent jobs, click Refresh from the toolbar.
