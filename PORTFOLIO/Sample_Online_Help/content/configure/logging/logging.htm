Logging

You can create logs for various events.

Viewing Logs
1. To view a log, log into the Configuration Console as administrator.
2. Open the Logs workspace. The list of logs appears.

Logs Workspace

3. Double-click an entry to view the log details.

Log Details

Logging Configuration

To access the Logging Configuration page, log into the Configuration Console as an administrator, then select Monitor > Application Logs > Logging Configuration.

Logging Configuration Menu

The Logging Configuration page appears listing the default log settings.
