Logging Out

When a user closes the last browser tab or window, then launches the browser again, they are allowed to resume their session without having to log in again.

Firefox. As of HEAT Release 2014.3, when a user closes the last browser tab or window, the active session is stopped. This enables other users to access the application without compromising licensing limitations. This is not possible for Internet Explorer or Chrome, at this time.
