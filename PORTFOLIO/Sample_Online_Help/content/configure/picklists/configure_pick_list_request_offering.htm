Configuring a Pick List for a Request Offering

This example shows how to configure a pick list for a request offering.

For a pick list to constrain the value based on another parameter in request offering, it must have special value "{0}" as a constraint value in the pick list definition.
1. From the Configuration Console, choose Pick Lists, and select a pick list. For example, select Employees. The Validation List details appear.

Choosing a Pick List

2. At the bottom of the window, click Add to add a line under Constraints.
3. Enter the values as illustrated:

AND Department Equal to {0}

Add Constraint

The other parameter that constrains this list (such as department) must be a drop-down list control.
4. Click Save.
