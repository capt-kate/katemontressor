Defining the Login Credential Priority

Use the Login Priority page to define which login credentials to use first when logging in.

1. Within the Configuration Console, open the Login Priority workspace.

By default, the Try internal login before external checkbox is selected, directing HEAT to use internal login credentials before attempting to use external authentication credentials.

2. To use external login credentials first, disable this option.
