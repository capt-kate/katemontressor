Service Desk Configuration

The Service Desk Configuration Wizard is part of the Application Setup Page. Use this wizard to define the access and workflow for information collected and acted upon by your company's service desk.

Process Wizard: Service Desk Configuration

User Interfaces

Use this page to define which roles have access to which modules and workspaces.

1. Select a role from the left column.  
2. From the right column, click Edit next to the workspace to which to provide access.
3. To enable the role to use the module or workspace, check Available. Clear the checkbox to hide the module or workspace for the role.
4. To provide a module or workspace button for the role, check Initially visible.
