Default Grid Settings

Use the Grid Setting page to define the default number of rows (records) displayed in dashboard and child panel grids. You can quickly change how the user interface and dashboard appears with this control.

1. Within the Configuration console, open the Grid Settings workspace. The Grid Settings page appears.

Grid Settings Page

2. Enter information into the fields.
--fields--
Field Description
Page Size for Workspaces The number of rows to automatically appear.
Use Best Fit

Allows the pages to fit the best way that they can within each workspace. If you check this option, the systems sets the Page Size for Workspaces field to zero.
Page Size for Dashboards The number of rows to automatically appear in dashboards.
Page Size for Popups The number of rows to automatically appear in pop-ups.
Page Size for Child Panels The number of rows to automatically appear in child panels.
3. Click Save and Update All Grids to save your settings and update all grids in all business objects. You can then apply custom settings to individual grids. Users will see the updated settings when they log in again.
4. Click Remove All User Options to delete all options set by individual users. Users will see the updated settings when they log in again.
