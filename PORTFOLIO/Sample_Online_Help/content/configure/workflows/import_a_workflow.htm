Importing a Workflow

You can import a workflow to use as a basis for other workflows or to duplicate a workflow.

1. From the workflow definition list, select the workflow to import into. The version list appears.
2. Check the workflow version you want to import to.
3. Click Import Workflow from the workspace toolbar. The navigation window appears.

Importing a Workflow

4. Navigate to and select your workflow XML file, then click Open.

The system imports the workflow and displays it as a new version on the version list.

Exporting a Workflow

You can export a workflow to use in other workflows.

1. From the workflow version list, check the workflow version you want to export.

Workflow Version List

2. Choose More Actions > Export Version. The export confirmation window appears.
3. Check Save File, then click OK.

The workflow XML file is saved to your local drive, usually to your downloads folder.
