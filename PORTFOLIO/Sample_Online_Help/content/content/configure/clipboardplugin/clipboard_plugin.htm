ClipboardData Plugin

Premise Users Only

The clipboard plugin enable users to copy and paste text between applications. Administrators can install the plugin on a server and it installs silently to clients.

1. On the server, place npClipboardDataPlugin.dll to C:\Program Files (x86)\Internet Explorer\plugins\.
2. Register using the following command:


regsvr32 /s "C:\Program Files (x86)\Internet Explorer\plugins\npClipboardDataPlugin.dll"

3. To ensure the plugin is installed correctly, go to Tools -> Manage Add-on.

ClipboardDataPlugin Installation Check

Users automatically have access without further action.
