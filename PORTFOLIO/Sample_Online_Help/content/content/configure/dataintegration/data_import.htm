Data Import

This option enables you to import data from a predefined source into the HEATSM database.

Set up a connection to the server from where the data is coming.

1. From the Data Integration Wizard (See Configuring Data Integration), click Start next to Data Import. The Connection page appears. Any configured connections are listed.

Data Integration Wizard

2. Click Add new. The Connection Setting page appears.
3. Select the connection type: Data Import or SCCM. Different configuration details appears for each type.
