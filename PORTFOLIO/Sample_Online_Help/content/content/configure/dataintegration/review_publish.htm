Review and Publish

You have three options to review or publish your data:

  • Click Preview to view the form with your field mappings. All records from the first batch appear showing the mapping configuration and the associated data. You can check these records for import errors before beginning import.
  • Click Publish & Run Now to run the import immediately and place it on the import schedule.
  • Click Publish & Test Run to run the import with only the first batch of records. A batch is 200 records. If errors or problems occur in this job, you can go back to your data and trouble-shoot before running the entire job.

Field mappings are automatically saved even if they are not immediately published.
After publish, the job runs based on the selected schedule.
