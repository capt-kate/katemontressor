Schedule Manager

This option does not appear if you select the manual option in the Connection Manager window.

The options in this page are defined in the Schedule Entry workspace. See Schedule Entry for more information.

Select one or more schedules. This is when the import is added to the queue for processing.

You should create multiple entries in the Schedule Entry table to run multiple imports of the same data each day.

If you are not the owner of a schedule you have selected, and the schedule is changed, your import time might be affected.
