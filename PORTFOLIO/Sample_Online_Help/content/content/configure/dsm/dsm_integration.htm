* * * New or Improved for 2014.2 * * *  DSM Integration

You can view the user DSM URL and client authentication key if you have administrator privileges.

This feature is only available in the new administator user interface, called the Configuration Console.
  • From the Configuration Console Settings pane, click Extend > DSM Integration. The system displays the DSM Integration page.

Sample DSM Integration Page

You can select the URL and client authentication key, then copy and paste to another application.
