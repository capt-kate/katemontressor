Viewing Configuration Items in the Managed Software Workspace

As a Service Desk Analyst, you can open the Managed Software workspace to view (but not change) managed software, its status, and other information.

The Managed Software workspace is for DSM administration and maintenance only. To fulfill software and provisioning requests, use the Incident or Service Request workspace, as described in Installing and Provisioning Incidents and Creating Software and Provisioning Service Requests.

Click the software and select DSM Managed Software to open the Managed Software workspace to view the following information:

  • Software synchronization and activation status
  • Software parameters
  • Whether a package is uninstallable
  • Whether approval is required for a package
  • Whether a package or set is published to Self Service
