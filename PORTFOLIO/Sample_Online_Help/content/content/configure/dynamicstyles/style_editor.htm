* * * New or Improved for 2013.2 * * *  The Style Editor

Administrators can create custom background and font colors within the Style Editor page. You can experiment with various color combinations before publishing to your Service Desk.

1. Within the Configuration Console, click Build > User Interface Styles > Style Editor. The Style Editor appears.
2. Select the styles that you want from the left pane, and preview the results in the main area.
  • Click Reset to revert your entry to default settings.
  • Click Choose to select a color from the palette.

Section Parameter Description
Logo

Click the folder to navigate to your logo image. For best performance, your logo should be less than 100 KB. The logo dimensions should fit within an 50px high by 200px wide area. Logos are not scaled, but rather are cropped to the center of the image.

 

You can define different logos for different roles. See About Roles
Common Color The background color of the home page.
Text The text color of all items in the home page, including dashboard panels.
Link The text color of any links on the home page.
Border The color of all borders on the home page, including borders around dashboard panels and borders under headings.
Selected Item The background highlighting color of selected text.
Over Item The background highlighting color of hovered text.
Panel Color The background color of all panels and dialog windows.
Header Color The background color of all panel headers and dialog windows.
Header Text The text color of all panel headers and dialog windows.
Header & Footer Color The main background color of the header bar and the footer. The header bar can be configured to show a gradient texture.
Center Color To give the header bar a gradient texture, enter a different color than the previous color. To give the header bar a solid texture, enter the same color value as the previous color.
Text The text color of the header bar area.
Link The color of links in the header bar.
Link Over The hover-over text color in the header bar area.
Workspace Selector Color The background color of the workspace.
Text The color of text in the workspace area, including inactive tab labels.
Active The color of the top of the active tab.
Buttons Color The background color of buttons, such as in the chat area.
Text The text color of buttons.
Highlighted Color The background color of a selected button.
Highlighted Text The text color of a selected button.
3. Click Apply to preview your settings. Your choices should automatically appear within 2 seconds without clicking this button, but you can expedite the process if needed.
4. Click Save to save your settings and publish to the application.
5. To test your settings, go to the Service Desk Console and select a different role. Be sure to return to administrator role before continuing to edit your styles.
