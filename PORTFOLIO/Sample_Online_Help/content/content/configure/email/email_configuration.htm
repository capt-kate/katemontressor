Email Servers

The configuration for the email service provider is stored as a service business object in the HEAT management database, while tenant-specific configurations are stored within each tenant database.

The email service sends emails using the individual SMTP configuration of the tenant.

Various options are available within the main Email Configuration window to fully customize your email settings. Tabs at the bottom of the window give access to many of these options.

The default port for sending surveys is HTTPS 443. However, port HTTPS 7075 is supported until further notice. You do not need to make any changes if you are using the global constant from central configuration.
Access Email Configuration
  • Log into the Service Desk Console as an administrator and open the Email Config workspace.

OR

  • From the Configuration Console Settings pane, click Servers > Configure > Email Configuration.

The Tenant Email Configuration page appears.

The default configuration includes a single inbox record. Use this inbox to create incidents from the client email of the user.

Configure the Outgoing and Incoming Email Servers

The email servers are usually configured during installation.

1. Log into the Service Desk Console as an administrator and open the Email Config workspace. The Tenant Email Configuration window appears.

Tenant Email Configuration Window

2. Enter information into the fields.
-fields-
Field Description
OUTGOING SERVER
SMTP host The host address, such as smtp.email.com
SMTP port The SMTP port number. The default is 25.
Authentication The authentication. The default is none.
Use SSL/TLS Uses SSL (secure sockets layer) or TLS (transport layer security).
Username A valid email address.
Password The password for the email address.
Send Mail Timeout The number of seconds to time out when sending an email.
Block Outgoing Email Blocks outgoing emails (only receive emails).
Email Address Override Overrides email address.
INCOMING MESSAGE HANDLING GENERAL RULES
Keep full incoming message as an attachment Puts incoming messages into attachments.
Store Journal Email As The format in which you want to store the emails: HTML or text.
3. Click Save.
