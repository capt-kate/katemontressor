Other Form Controls

Listed are the Other controls for business object forms.

Address Book

Enables the user to access the Address BookA HEAT object that stores names and email addresses from the employee and external contact objects. Users simply start typing a name in an address-book-enabled field, and a list of names and email addresses appears.. This control only applies to text fields.

-properties-
Property Description
Control Type Address Book
Width Enter a value for the control width (ems).
Height Enter a value for the control height (lines).
Visible Expression Accepts the expression $(true) to make the expression control visible, or $(false) to hide it from the user.
Style

Choose a display style for this control from the drop-down list.

  • Default. Label and cell default settings in accordance with the theme applied.
  • Highlighted Row. Enhances the cell with a highlight color.
  • Header. Applies larger text style to label for use as a form heading.
  • Bold Label. Makes the label bold.
  • Right Label. Sets the label flush right against the field/control.
  • Horizontal Rule Header. Sets the label against a shaded background.
  • Black Combo Box. Adds a combo box with a black background. Automatically makes the label bold.
  • Blue Text. Represents field entries in blue text.
  • Light Blue Text. Represents field entries in light blue text.
  • Light Blue Text Right. Field entries are represented in light blue text and right-aligned when displayed in the field.
  • Bold Highlighted. Represents field entries in bold text.
  • Bold w/Horizontal Rule. Represents field entries in bold underlined text.
  • Counter. Sets the cell as a counter. When a new record is added, the counter is incremented by 1.
  • Horizontal Rule. Adds a horizontal rule to a cell. Use rules to organize sets of fields and controls.
  • Vertical Rule. Adds a vertical rule to a cell.
  • Vertical Rule (long). Adds a vertical rule to a cell, with the top of the rule running into the cell padding.
  • VIP. Adds custom styling to a VIP cell.
  • Panel Header. Cell property creating the heading in a panel, which can serve as a component in a complex form.
  • Panel Footer. Cell property creating the footer in a panel form component.
  • Red Label. Field label is shown in red color font.
  • Panel Body. Cell property creating a body section in a panel form component.
  • Panel Body (No Border). Cell property creating a body section with no border in a panel form component.
  • Panel Body (First Row). Cell property creating a body section as the top row in a panel form component.
  • Therm labels top. Custom styling for the top portion of an escalation thermometer.
  • Therm labels bottom. Custom styling for the bottom portion of an escalation thermometer.
  • Link (button). Custom styling for a link button.
Style Rule

Enter an expression to be applied to this style, if needed. You can define what existing style (bold, header, etc.) to apply to a control based on a condition. See Style Rule Expressions in the Form Editor for more information.
Label Address Book
Label Expression

Enter an expression to be applied to this label, if needed. For example, on the KnowledgeSelfService form, the following expression has been added to the Details cells:

$(("Article ID: " + KnowledgeNumber))

This adds the label Article ID to the label and the Knowledge Base number assigned to this record.
Image Choose an image to be associated with this control. IE8 only supports images 32 KB or less.
Attachments

Allows attachments to be added, listed, and viewed on the form.

-properties-
Property Description
Control Type Attachments
Relationship Choose the related field from the drop-down list.
Can add True. Users can add attachments to the record.
Can delete True. Users can delete attachments from the record.
Size in Choose from the drop-down list: KB or MB.
Max files Enter the maximum number of files allowed to be attached to a record. For example, if you want no more than ten files to be attached, enter 10. Enter at least 1 to enable file upload.
Max files per upload Enter the maximum number of files allowed to be uploaded at one time. For example, if you only want six files to be uploaded during a single upload, enter 6. Enter at least 1 to enable file upload.
Max single file size Enter the maximum file size allowed to be attached. For example, if you only want to allow files of 2000 MB, enter 2000 in this field and MB in the Size in field.
Max summary files size Enter the maximum file size allowed for the summary file. For example, if you only want to allow a summary file of 1 MB, enter 1 in this field and MB in the Size in field.
Allowed file types

Type the file types (separated by commas but without spaces) you want to allow to be uploaded. The default values are:

jpg,png,xml,mp3,txt,doc,docx,xslx,xsl,pdf,zip,rar.
