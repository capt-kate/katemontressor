Grid Highlights

If you are an administrator, you can create custom styles to apply to grids. You can use these styles to highlight changes to individual records as they are listed in the grid when certain conditions apply. For example, you can have all unlicensed products appear in a list in a bold font with a red background. After creating a dynamic style, you can apply it to a field on a grid.

To apply colors to alternate records within a grid, see Formatting Rules.

Create a Grid Style
1. Within the Configuration Console Main Menu, choose Grid Highlights. The list of grid styles appears.

Grid Highlights Workspace

2. Click Add New .... The Dynamic Styles design page appears.
3. Under the Dynamic Style Definition section, enter the following:
Field Description
Name

A unique name that only contains alphanumeric characters. Spaces and special characters are not permitted.

Description

The purpose of the style.

Dynamic Style Definition

4. Drag and drop items from the Style Properties column onto the definition table, then enter standard CSS syntax to define each style. See w3schools.com for more information on CSS.
-styles-
Style Description

background-color

The grid cell background. Use standard CSS syntax, such as purple or #533b75.
border-color The color of the grid border.

color

The color of the grid text.
font-family The grid text font family.

font-size

The grid text font size.

font-style

The grid text font style, either normal or italic.
font-weight The grid text font weight, such as bold or normal. Note: 400 is the same as normal and 700 is the same as bold.
padding The amount of padding within a grid cell.

The Preview section shows the results of your custom grid style.

5. Click Save. The style appears on the page. You might need to Refresh the screen to see the style.
Apply a Format to a Grid

Once a custom style has been created, you can apply the style to a grid within a business object. See Formatting Rules. You can use the styles that you created with the grid highlighter on multiple grids.
