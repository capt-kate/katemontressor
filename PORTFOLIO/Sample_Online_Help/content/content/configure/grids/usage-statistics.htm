Grid Style Usage Statistics

You can see the business objects to which grid styles have been applied.

1. From the Configuration Console Main Menu, choose Grid Highlighter.
2. Click Get Usage Statistics. A list of the styles and where they are being used appears. If a style does not have a description, it is not being used.
3. Choose a style name from the list to view the colors and fonts defined.
