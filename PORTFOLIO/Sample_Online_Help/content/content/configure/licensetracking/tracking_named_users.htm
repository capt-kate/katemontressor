Tracking Named Users

Select Is Named User for each user to whom you are assigning a named user license. If a user is not specified as a named user, the user is treated as a concurrent user.

1. Within the Employee workspace, double click the employee record.
2. Select the Is Named User option.
3. Click Save.
4. Return to the list view and repeat steps 1 to 3 for each employee who has a named user license.
