Viewing the License Usage Report

1. Within the Configuration Console Main Menu, choose Report Editor. The system displays a list of reports.
2. Choose the License Usage report. The report appears in a new window showing the data from the time the report was last run.
3. In the Parameter Information section, enter the following:
Field Description

From Date

To Date

 The date and time range to report the data.
On Frequency The license usage frequency.
4. Click View Report. The report appears.

Sample License Usage Report

The license usage report contains the following information:

UTC date and hour

The date and time of usage

Named License

The number of named licenses available. Named license users can open multiple sessions using the same login information.

Named License in Use

The number of named licenses being used

Concurrent License

The number of concurrent licenses available

Total Concurrent License in Use

The number of concurrent licenses being used

Self Service License in Use

The number of Self Service licenses being used. This is tracked when the user logs into HEAT and selects the Self Service role.
