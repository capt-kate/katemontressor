Using the Translation Tool

HEAT provides a translation tool that lets you customize the text strings used in Self Service for each supported locale. You can use the translation tool to do the following:

  • Specify which user interface areas will use customized translation strings.
  • Update and modify the translation strings of one of the supported languages.
  • Import or export a spreadsheet containing some or all of a language's translation strings.
Do not attempt to add translation strings by importing a spreadsheet to which you have added rows or columns. See Import or Export a Translation Spreadsheet.
Use the Translation Tool
1. Within the Localization Tools page, click Translation Tool. The translation tool appears.

This option is automatically checked. This lists all strings, implemented and non-implemented.

  • If selected, the display area shows the implemented translation strings and all other translation strings in the database associated with the original-language term.

3. Click Search. The translation strings matching the criteria appear.
Edit the Translation Strings
1. In the third column, click a string to open it for editing, then enter or choose a new string.
2. Press Enter to record (but not save) the change. Changed strings that are not yet saved are marked for reference.
  • New strings are automatically indicated by logged-in user name and date/time created.
  • Modified strings are automatically indicated by logged-in user name and date/time changed.
3. Change additional strings as necessary.
4. To revert to the original set of strings prior to saving your changes, click Revert.
5. Click Save. The translation string changes are now implemented.

Optional. You can import the translation strings to a spreadsheet as described in Import or Export a Translation Spreadsheet.

Revert to Default Settings
  • To erase any strings you have input, click Revert. The strings are returned to original default settings.
Import or Export a Translation Spreadsheet

Use the Import from Excel and Export to Excel buttons to manage strings in a MS Excel spreadsheet. Importing all the strings to a MS Excel spreadsheet makes it easy to view and see which strings have not been translated. Also, use this feature if:

  • You have customized translation strings and need to back them up or restore them.
  • You have strings translated by an outside source.
When editing an exported translation string spreadsheet in MS Excel, edit the strings in the right-most column only. Making changes in other columns could result in database inconsistencies when you import the spreadsheet. Also, do not add rows to the database. Adding rows could result in database inconsistencies when you import the spreadsheet.

Export to Microsoft Excel

1. Within the Languages workspace, click Translation Tool. The translation tool appears.
If you make any changes, be sure to save before continuing to export.
2. Select the Properties to export.
3. Click Export to Excel.

Depending on your browser's configuration, either a spreadsheet is created in the browser's default download folder or you are prompted to specify a folder.

After the spreadsheet is exported, you can open it and modify the translations strings in the right-most column as necessary.

Import from Microsoft Excel

1. Within the Languages workspace, click Translation Tool. The translation tool appears.
2. Click Import from Excel. The File Upload window appears.
3. Navigate to the spreadsheet, select it, and click Open. The translation tool imports the strings in the spreadsheet and shows the import status when finished.
4. Click Search in the translation tool to update the list and show the imported translation strings.
