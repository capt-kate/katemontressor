CMDB Baseline Log

The CMDB Baseline log is included by default and lists the baseline IDs for specific objects.

View the Workspace
  • From the Configuration Console Settings pane, go to Monitor > Application Logs > CMDB Baseline Log. The list of IDs appears.

Sample CMDB Baseline Log

Delete a Log Item

Within the CMDB Baseline workspace, select the item that you want to remove, then click Delete from the toolbar.
