Integration Log

The Integration log is included by default and lists data integration import jobs. See View the Data Integration Log - File Integration.

View the Workspace
  • From the Configuration Console Settings pane, go to Monitor > Application Logs > Integration Log. The system displays a list of import jobs.

  • If you do not see any recent jobs, click Refresh from the toolbar.
