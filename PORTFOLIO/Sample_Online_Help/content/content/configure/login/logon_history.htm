Configure > Setting Up User Login > Logon History

* * * New or Improved for 2014.2 * * *  Logon History

You can get a list of when users log in from the Logon History page.

While logged in as administrator, do one of the following:

  • From the Service Desk Console, open the Logon History workspace.
  • From the Configuration Console Settings pane, click Monitor > Logon History.

The list of logins appears with the following information:

Column Description
User The username.
Logged from IP The IP address of the user.
Login Date/Time The date and time the user logged in.
Login Type The login type, such as new login.
Logout Date/Time The date and time the user logged out.
Logout Type The type of logout, such as explicit.
Role The role the user logged in as.
Authentication Type The authentication type used.
Last Request The date and time of the last login request.
Successful Login Success of the login: successful or unsuccessful attempt.
User Agent The browser and OS that the user was using.
Cancel User Sessions

You can simultaneously terminate multiple user sessions.

1. From the Logon History page, select the user sessions that you want to cancel.
2. Click Terminate User Session. The system disables this button if any of the selected user sessions are already terminated. A confirmation window appears.
3. Click Yes.

Terminate User Session
