* * * New or Improved for 2014.2 * * *  Adding Values to a Pick List

Values can be added to a pick list from the business object. Validation list values can also be added through the Validation Business Objects page. See Add Validation List Values.

1. Within the Configuration Console, open the business object containing the pick list object. For example, Change. Then from the Object Fields tab, click the field holding the pick list. For example, Category. The field information appears. See Using Fields
2. Next to the Pick List Name, click values. The values list appears.
3. Click Add. The value window appears.
4. Enter your information into the fields. The fields that appear might be different for each pick list. Translate the fields as needed.
5. Click Save, then click Close to close the window.
