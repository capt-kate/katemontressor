Login Priority

The Login Priority page is where you define which login credentials should be used first when logging in.

  • From the Configuration Console Main Menu, choose Security Settings > Login Priority.

By default, the Try internal login before external checkbox is selected, directing HEAT to use internal login credentials before attempting to use external authentication credentials. Disable this option to use external login credentials first.
