Branding

The Branding options only apply to the Self Service role. You can designate what Self Service users see, such as Logo or workspace items.

The Branding page is accessed from within the Applicaiton Setup Wizard (see Setting Up the Application), but its features can also be accessed from the Settings menu go to Configure > User and Permissions > Roles and Permissions > Self Service.

  • In the logo section of Your Organization, click Choose File to navigate to a graphic file that Self Service users will see upon logging in.

By default, eight tabs can be manipulated for display within the Self Service module.

  • Home
  • Social Board
  • Service Catalog: Lists available services and allows you to request a service.
  • My Items: Displays reported items and service requests.
  • Knowledge Center: Allows access to the organization knowledge repository.
  • Search: Defines search parameters.
  • Announcements: Displays system-generated announcements.
  • Alerts: Displays system alerts.
  • FAQ: Displays a list of frequently asked questions.

Click Edit for a tab to edit its display parameters. To change the tab name, edit its display text.

Check Available to make this tab available as a dashboard component for the Home page. Check Initially Visible to make this checkbox visible upon login.
