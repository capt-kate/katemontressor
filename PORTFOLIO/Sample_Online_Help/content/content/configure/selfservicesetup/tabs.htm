Tabs

Select the pages (user interfaces) to appear to Self Service users upon logging in.

1. Within the Configuration Console Main Menu, choose Self Service. The Self Service configuration page appears.
2. Under Tabs, click Get Started. The Tabs page appears.

The default tabs are:

Home The initial dashboard page.
Social Board The Social Board page, including Chat.
Service Catalog To create service requests.
My Items List of service items owned by the logged in user.
Knowledge Center To allow knowledge records to be created.
Search Search within multiple objects.
Announcements Receive announcements while in any page.
Alert Receive alerts while in any page.
FAQ To access the Knowledge database.
3. Click Edit next to each tab to configure the following options:
Title Enter the name of the tab that appears to the user.
Available Select this option to make this tab available to the user.
Visible Select this option to make this tab initially visible to the user.
Closeable Select this option to include a Close button (X), allowing the user to close the tab after they have opened it.

To change tab order

Click and drag the dotted area left of the tab name to reorder the tabs.
