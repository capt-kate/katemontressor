The Change Management Wizard

The Change Management wizard is part of the Application Setup Page. Use this wizard to customize how the Service Desk handles the change management process.

Process Wizard: Change Management

Classification

Define the types of changes within your organization. The default types are:

Type Description Role
Standard A pre-approved change that is low risk and follows a standard procedure. Use change request templates to automate your standard changes. Administrator
Minor A change with minor impact, which requires few resources and minimal review. Administrator
Significant A change with significant impact, which requires significant resources and pier review. Administrator
Major A change with major impact, which requires many resources, extensive planning and special approval. Administrator
Emergency A change that must be introduced immediately and which requires special processing. Administrator, Change Manager, Service Desk Analyst, Service Desk Manager
  • You can rename these types or add new ones.
  • To enable the change type, check the box preceding its name. You cannot delete change types, but you can disable them by un-checking the box.

This change type is enabled

Lifecycle

Select the actions to take at each stage of the Change Management lifecycle. The diagram shows the default lifecycle.

Action Description
When the change is "Canceled" Prompt Reason and Create Journal Notes when change is canceled
When the change is "Closed" Notify Requestor that change is Closed
When the change is "Deferred" Prompt Reason and Create Journal Notes when change is Deferred
When the change is "Denied" Prompt Reason and Create Journal Notes when change is Denied
When the change is "Implemented" Complete task when the change is Implemented Notify Requestor that change is Implemented (Preview Email)
When the change is "Pending Approval" Notify Requestor that change is Pending Approval (Preview Email)
When the change is "Scheduled" Notify Requestor that change is Scheduled (Preview Email)
  1. To add actions, click Add another Action (at the bottom of the page), choose the Action Type and associated Action, then click Save.
Review Board

Identify users to participate in your Change Approval Boards (CABs). Review the default CABs to determine their membership.

  • Click Add Group to add another group to the list.
  • Within each group, click Add Member, to add another member to the group.
  • Click Unlink to remove a member from the group.
Group name Description
Change Advisory Board CAB group for Non Emergency changes.
Emergency Change Advisory Board CAB Group for Emergency changes.
Management Committee Management Committee for Major changes.
