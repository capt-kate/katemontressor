The Request Management Wizard

The Request Management wizard is part of the Application Setup Page. Use this wizard to customize how the Service Desk will handle request management processes.

Services Provided

Use this section to manage the services provided by your IT department. You can customize this set or save them and return to add or update the services later.

Process Wizard: Request Management

1. Click Add Service.
2. Enter the Name and Description of the Service.
3. Click Save & Activate.

The category is added to the list and rendered active. By default, each Service is active and available for customization.

Click Edit to edit a service name or description.

Lifecycle

Select the actions to take at each stage of your Request Management cycle. If you have advanced lifecycle requirements, you can define them later.

Process Wizard: Request Management: Lifecycle

The default lifecycle settings are:

  • When the request is "Approval Rejected": Click the Cancel Outstanding Approval on rejection checkbox to cancel any outstanding approvals related to this request when the request status changes to Rejected.
  • When the request is "Canceled": Click the Cancel Outstanding Approval on cancellation checkbox to cancel any outstanding approvals related to this request when the request status changes to Canceled.
  • When the request is "Closed": Click the Cancel Outstanding Approval on close checkbox to cancel any outstanding approvals related to this request when the request status changes to Closed.
  • Click the Send Survey upon Service Request Closure checkbox to automatically send an email notification when the request is closed. The email contains a link to a customer satisfaction survey.
Offerings

Use this section to select request offerings to publish in your Service Catalog.

Process Wizard: Request Management: Offerings

  • Click Rename for a particular offering to create custom names.

By default, all offerings are activated for customer access. Click Deactivate to deactivate an offering for the entire organization.

  • Check Show offerings for deactivated services above the table to make inactive offerings visible to the customer.
  • Use the Edit Subscription link to manage the active offerings for each organization unit.
