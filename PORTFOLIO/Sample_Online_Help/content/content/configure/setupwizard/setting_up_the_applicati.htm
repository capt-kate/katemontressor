Setting Up the Application

You can initially set up your modules in HEAT on the Application Setup page (formerly known as Process Wizard). After initial setup, you can access various parts of Application Setup within the Configuration Console to modify your settings.

Before you initially set any application parameters, a Setup Wizard link appears on the HEAT pages. Once you have run the setup wizards and saved your settings, the Setup Wizard link no longer appears.

For Premise users, this page is not the same as the Application Setup Wizard used to set up the HEAT Service Management system.

Application Setup Page

1. To use the setup wizards, within the Configuration Console from the Settings header, click Search . A list of configuration workspaces appears.
2. Click Application Setup. The Application Setup page appears.

Application Setup page

Within the Application Setup page, a list of available wizards appears. You can run each wizard in the order provided, or skip around and enter information as needed.

  • To start a wizard, choose an item from the left menu, then click Start Section.
  • To skip a module, click Accept Defaults and Skip Section.

After initially running the wizards, you can go back and change the settings in these pages.
