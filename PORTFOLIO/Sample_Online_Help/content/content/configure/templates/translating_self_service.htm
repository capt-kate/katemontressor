Translating Self Service Templates

Self Service templates are available for Self Service users to use from the Self Service module. You can edit any of the English field values and provide corresponding values in the supported languages.

1. Create a template quick action or edit a Self Service template quick action.
Notice that the publish to action for all the Self Service templates are set for the Self Service users: Self Service, Service Desk Analyst, Service Desk Manager, and Service Owner.
2. Click Edit translations.
3. Enter the translated text corresponding to the English field values for the language that you want to configure.

These fields contain the translated values when the application appears in the set language.
