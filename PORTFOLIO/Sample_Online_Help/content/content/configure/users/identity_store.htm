Using the Identity Store

The Identity Store Workspace

The Identity Store workspace lists all users who have login access to HEAT . These users have login IDs and passwords.

View the Identity Store List
1. Log into the Service Desk Console as an administrator, and open the Identity Store workspace. A list of identities appears.
2. Double-click a record from the list to see details.
Show Identity Details

Within the Identity Store workspace, select a user name from the list. The user details appear in the bottom panel of the window.

For more information, select a user from the list, then click Show Identity Details from the toolbar. The Employee workspace opens and the user details appear on the page. See Using Employees. Edit employee details within the Employee or External Contact workspace.

Export an Employee List
1. Within the Identity Store workspace, click Export Employee LIst from the toolbar. The export confirmation window appears.
2. Click Yes to export the records, or No to cancel the export and dismiss the window.
3. If you click Yes, a second confirmation window appears. Click Yes to continue to export, or No to cancel the export and dismiss the window.

Depending on how many records you have, the export might take several minutes, tying up your computer and possibly the database.

4. If you click Yes, then the Open Export window appears.
  • To open the Export file in MS Excel, select Open.
  • To save the file to your local computer, select Save File. The file is saved to your Downloads folder.
  • Click OK.
