Using Teams

The Standard User Team Workspace

Teams are pools of employees who can be assigned tasks within HEAT, such as Advisory or Review boards. An employee cannot be assigned a task without being part of a team.

You can manage teams associated with an Employee. Teams are pools of employees who can be assigned tasks within HEAT, such as Advisory or Review boards. An employee cannot be assigned a task without being part of a team.

View the Team List
1. From the Configuration Console Settings pane under Configure, go to User and Permissions > Teams to view a list of Standard User Teams.

OR

Log into the Service Desk Console as an administrator and open the Employee Team workspace.

2. Double-click a team name to view its properties and members.
Create a Team

You can add teams ad hoc from the Employee page (see Using Employees), or use the Team page to link employee accounts to existing teams.

1. From the Standard User Team workspace, click New Standard User Team. A blank Standard User Team page appears. The following default tabs appear (results may vary depending on your system setup):
  • Details
2. Enter information into the fields.
-fields-
Field Description

Team

Enter a unique name for the team.

Service Desk

Check to add this team to the Service Desk workflows.

Team Email

Enter the name of the email team which relays messages to each team member.

Team Manager

Start typing a manager name to automatically search from the database.

Team Manager Email

Enter the email address for the team manager.

Average Cost Per Minute

Enter a number.
3. Click Save from the toolbar. The team now appears on the team list.
Create an Ad-Hoc Team
1. Within an Employee record, select the Team tab. The team list appears.
2. From the toolbar, click Link. A list of teams appears.
3. At the bottom of the window, click Add. The New Standard User Team window appears.
4. Enter your information into the fields. See Create a Team.
5. Click Save. The team now appears on the team list.
6. Select the team from the list, then click Select. The team is added to the Employee team list.
7. Click Save from the toolbar.
Modify a Team
1. From the Standard User Team workspace, double-click the team you want to modify.
2. Edit the team using the options described in Create a Team.
3. Click Save from the toolbar.
Delete a Team

Any team can be deleted, including HEAT default teams.

Deleting a team might cause unexpected results. Delete the team from a workflow before permanently deleting the team here.
  • From the Standard User Team workspace, select the team you want to remove, then click Delete from the toolbar.
Add Users to a Team

You can add new users, link users, and unlink users from the Standard User Team page or under the Employee section. See Create an Employee.

1. From the Standard User Team workspace, under the Employees tab, click Link. A list of users appears.
2. Select the users you want from the list. Ctrl + click to select multiple users.
3. Click Select. The users appear in the member list.
4. Click Save from the toolbar.

Unlink Users from a Team

  • Select a user name from the Member list, then click Unlink. The user no longer appears on the team list.

Add a New User to a Team

  • Click New Employee. The New Employee page appears. See Create an Employee for information about this form.
