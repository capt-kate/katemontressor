* * * New or Improved for 2014.1 * * *  Configuring Voice Integration

To enable a Start IPCM Voice Activity quick action, you must first configure HEAT to connect to IPCM.

The IPCM server uses the following settings:

Premise Configuration

Contact your FrontRange representative for information on implementing this feature. See Further Information for contact details.

Cloud Configuration
1. From the Configuration Console, choose Extend > Voice Integration > Cloud Configuration. The system displays the Cloud Configuration page.
2. Enter information into the fields.
-fields-
Field Description
IPCM Administrator contact information
Name A valid user name.
Location The physical location of the administrator. The system automatically generates this if the user is in the employee records.
Email A valid email address. The system automatically generated this if the user is in the employee records.
Phone A valid phone number. The system automatically generated this if the user is in the employee records.
My Voice Integration Information
Connect IPCM SIP Soft Switch to my SIP gateway or PBX at

Adds static registration settings for the trunk:

  • IPv4 Address: A public IP network address.
  • Port: A public IP network address.
My SIP gateway or PBX will register with the IPCM SIP server using

Adds dynamic registration. The SIP Server accepts REGISTER requests to this trunk, and prompts you for your user name and password.

  • Identity: The trunk name.
  • User Name: The authentication user name.
  • Password: The authentication password.
Additional Configuration
S2EC Box Access
IPv4 Address The public IPv4 address of the IPCM HEAT Enterprise Connector server.
Message Server Port The public TCP port of the IPCM Message Server service (default is 61000).
Reset Password Service Port The public TCP port of the IPCM Reset Password service (default is 62000).
Voice Recording Storage
Host UNC Path The UNC path to the voice server storage directory. This location must be on or accessible by the IPCM HEAT Enterprise Connector. We recommend having a dedicated voice recording storage machine on the customer network, separate from the S2EC machine.

The UNC path is relative to the IPCM HEAT Enterprise Connector, rather than the IPCM server.
HTTP Base Path The HTTP (or HTTPS) URL used for internal access to the voice server UNC directory. By default, the IPCM HEAT Enterprise Connector machine serves the recordings from the local voice recording storage directory under the IPCM installation root. Define the virtual directory to access the UNC host path setting.
Reset Password Service Authentication
UserName The web service user name needed to authenticate an IPCM Reset Password web service request.
Password The reset password needed to authenticate an IPCM Reset Password web service request.
FrontRange Voice (IPCM) Access Information
IPCM Web Configuration URL Shows the URL for the IPCM web configuration file.
IPCM Web Statistics URL Shows the URL for the IPCM statistics and metrics engines.
IPCM S2EC Installer Link Shows the URL for the HEAT Enterprise Connector installation program.
IPCM Client Installer Link Shows the URL for the IPCM client installation program.
IPCM SIP Soft Switch Shows the host name and port number for the SIP soft switch.
IPCM Server

Shows the host name and port number for the IPCM server.
3. Click Save Settings.  
Hybrid Configuration
1. From the Configuration Console, choose Extend > Voice Integration > Hybrid Configuration. The system displays a list of hybrid connections.
2. Double-click the connection that you want to use. The system displays the IPCM Integration page.
3. Enter information into the fields.

Voice Hybrid Configuration Page

-fields-
Field Description
IPCM Server Host Name A domain name or IP address that can be resolved from the HEAT server.
IPCM Server Port The port number is set to 5743 by default.
IPCM Web Services User ID A valid user ID configured in IPCM. No specific role is required.
IPCM Web Services Password A valid password for the user ID configured in IPCM. No specific role is required.
IPCM Web Services Port

Set to match the web services port set in the IPCM management portal

(System Configuration > Voice Server > Advanced Configuration > App Server).

This is set to 2323 by default.
IPCM Web Service URL The URL to the IPCM web service.
Voice Recording UNC Path The UNC path to the voice recording application.
Voice Recording HTTP Path The HTTP path to the voice recording application.
Customer PBX Host The customer PBX host.
Customer PBX Port The customer PBX port.
Register to Customer PBX Registers the connection to the customer PBX.
Trunk Registration Number The trunk registration number.
Trunk User Name A valid user name for the trunk.
Trunk Password A valid password for the trunk user.
4. Under the Employee tab, click Link to link employees to this connection.
5. Click Save.

You can now define the Start IPCM Voice Activity Quick Action.
