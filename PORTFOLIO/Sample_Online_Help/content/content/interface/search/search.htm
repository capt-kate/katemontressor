Search

Search is available in most workspaces to help users to easily find records. Advanced searches can include expressions to fine-tune your search criteria. You can create a saved search, keeping frequently used criteria handy, and even designate a saved search as a favorite.

The search area has three main parts:

  • The Saved Searches link, text box, and drop-down list. You can choose from any search profiles available to the current record type and user role.
  • The Advanced Search link. Click to view the Advanced Search window.
  • The Search text box. Enter search criteria. You can search text based on the localized value.
