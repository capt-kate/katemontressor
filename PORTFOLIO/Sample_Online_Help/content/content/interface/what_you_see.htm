Navigation

After logging into HEAT, you see the Home page, which is one of the Dashboards. Different roles see different dashboards on their Home page.

Mobile Devices

HEAT can be used on most mobile devices, including tablets and smart phones, through a browser. All procedures described in this help apply to mobile devices.

iPad and iPhone

HEAT is available as a free app from the Apple store to enable iPad and iPhone users to interact with the database. See Mobile Layouts.

 

Main Workspace Parts

Top Header Bar

The top header bar has the following components:

  • Logo

  • Social Board button

  • Workspace selectors
  • More workspace selectors
  • User log in name and role
  • Logout button
  • Settings menu (Change Locale, Configure Application)
  • Online help

 

Other areas include the following:

  • Top tab bar, with workspace tabs

  • Dashboard selector (on Home workspace)

  • Watch list
  • Page title

  • Search area
  • Grid toolbar, with Action Menu
  • Grid
  • Record preview

 

Workspace Footer Bar

The workspace footer bar has the following components:

  • Page size selector
  • Page number selector
  • Previous and next arrows, first page and last page buttons
  • Refresh button
  • Record details
