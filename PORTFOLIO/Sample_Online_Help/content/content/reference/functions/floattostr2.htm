FloatToStr2

Converts a number value to a text value with two decimal places.

Syntax

FloatToStr2(number)

Enabled for

Search/Dashboard No
Business Rules No
Quick Actions Yes
Workflow Objects No
Reports No

Parameter

number Number value to convert to a text value.

Return Value

Text value.

Example

$(FloatToStr2(999.123))

The above example returns a text value of "999.12".
