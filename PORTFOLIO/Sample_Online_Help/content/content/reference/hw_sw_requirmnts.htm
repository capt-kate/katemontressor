Hardware and Software Requirements

System Requirements

System requirements vary by implementation. Problems can occur if you do not follow the recommended system requirements exactly. For the latest requirements, contact FrontRange Support.

Application Server

  • Microsoft .NET Framework, version 3.5 SP1
  • Microsoft Internet Information Services (IIS)
  • Server 7.0 (Windows 2008)

Browser Support

  • Firefox 3.6.x plus
  • Google Chrome
  • Microsoft Internet Explorer 9.x plus
  • Microsoft Internet Explorer 8.0 (IE8) is known to perform slowly with JavaScript-based applications like HEAT. IE8 also does not support CSS3. HEAT supports IE8, but we cannot guarantee the speed at which it renders the application. See details at: http://support.frontrange.com/common/files/support/heat_cloud/user/index.html#reference/hw_sw_requirmnts.htm?Highlight=Microsoft.

For optimal performance, we suggest you use a newer version of Internet Explorer or a different browser.

HEAT Database Server Requirements

Hardware

Component

Recommended

CPU
  • 2 core minimum
  • 4 core recommended for heavy workflow/escalation activity

RAM
  • 8 GB minimum
  • 12-16 GB recommended depending on transaction rates and workflow/escalation complexity

Hard Disk

10 GB minimum free space.

Note: This disk space requirement is the minimum. For proper disk space sizing, use these guidelines:

  • HEAT Application Database: 500 MB
  • Average disk space requirement per record (incident, problem, change, configuration): ~200 KB
  • Average disk space requirement per record (CI): ~500 KB
