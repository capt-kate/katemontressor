* * * New or Improved for 2013.2 * * *  Linux/Unix Support

Linux and Unix drivers are available for download from our support site. You must be a registered user to obtain these files.

1. Go to http://support.frontrange.com/support and click Technical Support.
2. Log in, then from Drivers & Downloads, choose HEAT Service Management.
3. Select the driver you want.

Each ZIP package contains an INSTALL.TXT file that describes the files that are included, which files you need, where to install the files, and the commands needed to start up and shut down the client.
