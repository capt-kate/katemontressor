Notification Workflow Block

Use the Notification block to define settings for email notifications. You can create automated messaging within the body of the email message as well as designate the recipient individuals and teams.

Notification Workflow Block

Exit Ports

sent  

Properties

Title Enter a unique name for the object.
Logging in Journal Check to store the text of the email in the Journal.
From Enter a valid sender email address or click Address BookA HEAT object that stores names and email addresses from the employee and external contact objects. Users simply start typing a name in an address-book-enabled field, and a list of names and email addresses appears..
To Enter a valid recipient email destination address or click Address BookA HEAT object that stores names and email addresses from the employee and external contact objects. Users simply start typing a name in an address-book-enabled field, and a list of names and email addresses appears..

CC

BCC

 Enter a recipient email address to copy (CC) or blind copy (BCC) or click Address BookA HEAT object that stores names and email addresses from the employee and external contact objects. Users simply start typing a name in an address-book-enabled field, and a list of names and email addresses appears..
Plain Text Check to send the email in plain text format (no HTML formatting).
Subject Enter a brief subject line.
Body Enter and format the text of the automated message.
Description Enter a generic description, or an expression that takes the text from a field. For example, $(Details).

Actions

  • Click Save to commit your settings.
