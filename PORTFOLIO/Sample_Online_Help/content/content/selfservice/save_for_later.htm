* * * New or Improved for 2014.1 * * *  Save for Later

You can create a service request and save it without issuing a notification. You do not need to enter all the required information at this time.

1. At the bottom of the request page, click Save for Later.

Save for Later

2. Click Cancel to return to the Service Catalog.

The item appears on your list of My Open Items. You can complete this form at a later time and submit the request. The workflow starts when you submit the request.

Save for Later is only supported for the Self Service role. This record appears to Service Desk Analyst users in draft status. The system does not send a request notification until you submit it.
For upgraded databases: You must update the existing workflow for a new request notification so that emails are not triggered when the request is saved in draft status.

Delete a Saved Request

The system displays any requests that have been saved for later in a pane on the right side and displays a Pending Requests link in the workspace search area. See Pending Service Requests.

  • Click Delete next to a pending request to permanently remove it.
