Timezone Examples

Create a Timezone
All timezones are included in HEAT by default. You can create your own timezone or edit an existing timezone.

This example shows how to create a timezone.

1. Log into HEAT as an administrator and access the Configuration Console.
2. From the Service Desk Configuration toolbar, click Business Objects. The Business Objects overview page appears.
3. Click Incident.

The Incident - Business Object Details page appears.

4. From the toolbar, click Object Fields. The Object Field list appears.
5. Click Add New.

The Add New Field page appears.

6. Click Date/Time.

The Property Details page appears.

7. Enter the following properties:

Property

MYT

Display Name

Malaysia (Kuala Lumpur)

Description

time zone for Kuala Lumpur

Type Specific Information

Ensure all options are selected: Hold Date, Hold Time, TimeZone
8. At the bottom of the page, click Add This Field. The field appears in the field list.
9. Click Save. This adds the field to the business object.
10. Go to the Service Desk Console.
11. From the workspace selector, choose More > Timezones. A list of timezones appears.
12. From the list, double-click the timezone you just created (Malaysia). The timezone edit page appears.
13. Select the following:

Is Enabled

Select to allow it to appear in the list control

Time Diff

Enter 480 (Kuala Lumpur is + 8:00 UTC).

Sort Order

Enter 2
14. Click Save.
15. Click List View. Your timezone appears in the list.

The new timezone now appears on any forms to which it has been added.

Add a Timezone to a Form

This example shows how to add a timezone control to a form.

1. Within the Configuration Console toolbar, select Business Object, then choose Incident.
2. From the toolbar, select Forms. The available forms appear.
3. Select Incident (Incident.Approval). The form workspace appears.
4. Open the Incident list, then open Fields. The list of fields appears.
5. Scroll down to Malaysia, then drag it onto the form.
6. Select the field in the form to activate properties box.
7. In the Width property, enter 55. The Your Local Time field widens.
8. Click Save and close your tabs.
