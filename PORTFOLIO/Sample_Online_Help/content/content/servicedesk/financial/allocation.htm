Allocation

Allocation shows subscribed services consumed and is automatically calculated based on the schedule Service Allocation Calculation Daily Schedule. For example, Data Service might provide 25% of it's total services to CRM Service. This cost is automatically calculated each month for CRM Service. The schedule gathers information daily, but provides data at the end of the month.

CRM Service Costs - 25% of Data Service costs for May (7,597.50) are charged to CRM Service

Establish Service Allocation

You can set the percentage of services consumed for each service provided. You can allocate as many services as you want. When the Service Allocation Calculation Daily Schedule is run, the total is listed under the Cost Item tab of the dependent service. The schedule is run daily (hence the name), but only the monthly totals appear under the tab.

1. Log into the Service Desk console as Administrator, open the CI Service workspace, then open the service record to which you want to set allocations.
2. From the Allocation tab, click New ITFM Recurring Service Subscription. The New ITFM Recurring Service Subscription window appears.
3. Enter information into the fields.
-fields-
Field Description
Status

Choose a status for the allocated service from the drop down list. Only the Active status is used in cost calculations.

  • Pending
  • Active
  • Paused
  • Obsoleted
Consuming Service Enter the name of the consuming service, or choose a service from the list.
Supporting Service This value is automatically entered corresponding to the service record you are in, but you can change this if needed.
Allocation (%) Enter a whole number for the percent of use the consuming service provides to the supporting service.
Default Cost Driver Enter the default cost driver for this service, or choose a cost driver from the list.
Notes Type a description for this transaction.
4. Click Save. The window closes.
5. Repeat this process for additional consuming services.
6. Click Save from the toolbar.
Setting the Service Allocation Calculation Daily Schedule

This schedule is not set by default, to prevent it from running for users who don't need it.

1. Log into the Service Desk console as Administrator, open the Schedule Entry workspace, then open the Service Allocation Calculation Daily Schedule. The form appears.

Service Allocation Calculation Daily Schedule

2. Enter information into the fields.
-fields-
Field Description

Start Time

End Time

 Set the times you want the schedule to start and end each day.
Start Date Set the date you want the schedule to begin.
No End Date Check this option to continuously run this schedule. By default, this option is not checked to prevent the schedule from running for users not needing it.

You can modify other fields as needed, or leave them as set.

3. Click Save from the toolbar.
