Using the Export to Excel Quick Action

Microsoft Excel does not support exported expressions.

You can use the Export to Excel quick action to export all or some of the field values from a business object to a Microsoft Excel spreadsheet. For example, you can export some of the field values from a Change record to an Excel spreadsheet.

1. Open the Quick Actions Center for the Change business object.
2. Click the Create New Actions tab, and then click Export to Excel from the list of options
3. Enter the following information:
Field Description
Set action name Enter a name for this quick action.
Assign a category Select a category from the drop-down list, for example, Form Actions.
Publish to Select the roles which need access to this template (administrator, Change Manager, for example).
4. Edit, add, or leave the Column Header and Fields to Export values as is.
5. To import relationships:
  • Click Edit.
  • From the Fields window, double–click Relationships.
  • Double–click the Relationship business object, then double–click Fields.
  • Select the field to import from the Relationship business object and click OK.
6. Click Save to save the quick action.
