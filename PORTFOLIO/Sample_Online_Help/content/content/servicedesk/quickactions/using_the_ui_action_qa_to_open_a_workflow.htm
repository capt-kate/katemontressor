Using the UI Action Quick Action to Open a Workflow

The following example shows how you can use a UI Action quick action to show the approval process of a Change.

1. Within a Change object, from the toolbar, select Action Menu > Edit Actions. The Quick Action page appears.
2. From the Create New Actions tab, select UI Action from the list of options. The UI Action interface appears.
3. Enter your information into the fields.
-fields-
Field Description

Set action name

Enter a name for this quick action.

Assign a category

Select a category from the drop-down list.

Scope

Type the command: CurrentTab:ObjectWorkspace:ObjectView.

Publish to

Select the roles which need access to this template (administrator and Change Manager, for example).

Command id

Type ShowWorkFlow.

Confirmation

Type null.

Command Data

{

       ObjectType:"Change#"

       ,ChildObjectType:"Change#"

       ,NewObjectType:"Change#"

}

4. Click Save to save the quick action.
