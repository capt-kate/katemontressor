Using the Update Object Quick Action

You can use the Update Object quick action to update certain fields on a business object. For example, you can manage email statuses, as described in Email Examples.

You can update the status field on a Change record to Implemented, as described below.

1. Open the Quick Actions Center for the Change business object.
2. From the Create New Actions tab, choose Update Object from the list of options.
3. Enter your information into the fields.
-fields-
Field Description
Set Action Name Enter a name for this quick action.
Enter action description Optional. Enter a description of what this quick action will accomplish.
Assign a Category Select a category from the drop-down list, For example, Form Actions.
Publish to Select the roles which need access to this template. For example, administrator, Change Manager.
Hide when

Enter a command to hide this quick action under certain conditions.

For example, if you do not want the quick action to be enabled when the Change record is in Logged or Closed state, enter the following command in the Hide when field: $(iif( (((Status == "Closed") || (Status == "Resolved")) || (Status == "Logged")), true, false))
4. In the Apply Update to section, make a selection.

By default, the action is applied to the Change (selected object). However, you can click the Child Object radio button to indicate this action should be run against a relationship. Then select a child object from the drop-down list.

5. Complete the Field Name and Field Value sections to customize the update object action:
  • For Field Name select Status.
  • For the corresponding Field Value, select a value for the Status field, which in this example is Implemented.
  • Click Add to add a row.
  • If you want the Last Modified By field to update, select LastModBy for Field Name.
  • For the corresponding Field Value, click Search next to the field to open the Edit Expression window, then double-click the Functions folder, and double-click CurrentLoginId to display $(CurrentLoginId()) in the left.
  • Click Save.
  • Click Add to add a row.
  • If you want the Modified On field to update as well, select LastModBy for Field Name.
  • For the corresponding Field Value, click Search next to the field to open the Edit Expression window, then double-click the Functions folder, and double-click CurrentDateTime to display $(CurrentDateTime()) in the left.
  • Click Save.
6. Click Save to save the quick action.
