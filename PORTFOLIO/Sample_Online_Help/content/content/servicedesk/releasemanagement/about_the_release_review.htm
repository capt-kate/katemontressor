About the Release Review Board

After each milestone is completed, the Release Review Board is notified to do a post implementation review of the milestone. See Working with Milestones.

By default, the approval workflow is set up only for major types of releases created by templates. See About Milestone Workflows.

The Release Review Board is a defined container for approving post-milestone implementation tasks.

View the Release Review Board
  • Log into the Service Desk Console as the Release Manager, then open the Release Review Board workspace. The default group appears.

Release Review Board Workspace

Double-click a record to view details.

The Release Review Board workspace is only available if your role has access.
