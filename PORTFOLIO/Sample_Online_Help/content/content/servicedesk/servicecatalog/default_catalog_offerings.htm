Default Catalog Offerings

New Application Access Request

Request installation of or access to a new software application.

Tab Name Field Description
Service Options Date Required

Enter the date by which the access to the application must be available.
Department Enter the department of the person making the request. Choose from the drop-down list.
Requester Enter the name of the person making the request. Choose from the drop-down list.
Application Details Additional Productivity Tool Type the specific productivity applications requested.
Performance Evaluation Tool Check if the requested application is for testing performance. For managers only.
Financial Application Modules Type the specific financial applications requested.
ERP Application Modules Type the specific ERP applications requested.
CRM Application Modules Type the specific CRM applications requested.
Additional Applications Type other specific applications requested.
Web Site Move Request

Request that a web site be moved to another server.

Tab Name Field Description
Requester Details Department Enter the department requesting the move. Choose from the drop-down list.
Contact Name Enter the contact person for this site. Choose from the drop-down list.
Contact Email Enter the email address of the contact person.
Contact Phone Enter the phone number of the contact person
Web Site Details Existing Site Location Enter the URL of the existing site.
New Site Location Enter the URL of the new site.
Requested Live Date Enter the date you want the site to go live. One week lead time is required.
