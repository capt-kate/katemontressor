Canceling a Service Request

Service requests cannot be deleted, but they can be canceled. This procedure changes their status to canceled, but keeps the service request in the system.

Once canceled, a service request can not be edited.

Cancel a Service Request

1. Within a service request record, from the Status menu, select Cancelled.
2. Click Save from the toolbar.

The service request now appears in the List View as Cancelled.

Cancel a Service Request from Self Service

1. Log into the Service Desk Console in the Self Service role and open the My Items workspace. A list of items appears.
2. Double-click the service request item you want to cancel. The Item Details page appears.
3. Click Cancel Request. The Cancel Service Request confirmation window appears.
4. Click OK to continue to cancel the request, or Cancel to dismiss the window without canceling the service request.

When you return to list view, the service request status shows Cancelled.
