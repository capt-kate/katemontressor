Fulfillment Item Packages

A fulfillment item package defines a requestable item, the tasks that are performed to fulfill the item, and which organizational units are entitled to select the item. When a Service Catalog user submits a request for a fulfillment item package, fulfillment items based on the fulfillment item package are created and added to the resulting Service Request.

Each item can be displayed as a selectable option to a Service Catalog user. You must create a fulfillment item package for each item that you want to display as a choice to a Service Catalog user.

A fulfillment item package can contain a combination of selectable items from task catalogs and software installation catalogs. For example, a fulfillment item package named Required Software could contain two software installation catalogs (each for the automatic installation of a different DSM-managed software package) and one task catalog for an SDA to manually give a user permission to access the location where the software resides. If a Service Catalog user selects Required Software in a Service Catalog request form, the tasks defined in the fulfillment item package are created upon approval of the request.

Create a Fulfillment Item Package
1. Log into the Service Desk Console as a service owner or administrator, and open the Fulfillment Item Package workspace.
2. From the toolbar, click New Fulfillment Item Package. A blank record appears.
3. Enter information into the fields.
4. From the Organizational Unit tab, click Link. A list of organizational units appear.
5. Choose an organizational unit from the list, then click Select. The organizational unit appears under the tab.
6. Click Save on the toolbar to commit your settings.

Administrators can also create Fulfillment Item Packages when creating organizational units (see Organization Chart).

Link Organizational Units and Task Catalogs

Use the Organizational Unit and Task Catalog tabs to link organizational units, task catalogs, and software installation catalogs to this fulfillment item package.

Organizational units that you link here are entitled to select the service offered in this fulfillment item package.

Task catalogs and software installation catalogs that you link here define the tasks for fulfilling the items that are included in this fulfillment item package.

1. Within the Fulfillment Item Package record, choose the Organizational Unit tab.
2. Click Link from the tab toolbar.
3. Select the organizational unit you want to link, then click Select.
4. Click Save from the toolbar.
5. Choose the Task Catalog tab.
6. Click Link from the tab toolbar.
7. Select the task catalog or software installation catalog you want to link, then click Select.
8. Click Save from the toolbar.
