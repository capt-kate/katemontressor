* * * New or Improved for 2014.1 * * *  Using Multiple Currencies

You can create different prices for users depending on where the customer is located. You can combine different currencies within a single Service Catalog offering.

For example, if a user creates a Service Catalog request, the prices will automatically reflect the location of the user, based on the defined Location within the Employee record.

Create an Ad-Hoc Price for a Request Offering
1. Log into the Service Desk Console as an administrator, and open the Price List workspace.
2. From the toolbar, click New Price Item. A blank price form appears.
3. Enter the following into the fields:
Field Description
Enabled Check this option.
Item Name Blackberry
Price Enter 99 for value and USD for currency type.
Cost Enter 25 for value and USD for currency type.

Sample Request Offering - Blackberry Price

4. Click Save from the toolbar.
5. Now open the Request Offering workspace and select the New Smartphone Request. The Define Request Offering page appears.

Notice that Use Price List is enabled. (See Create a Request Offering.)

Sample Request Offering - New Smart Phone Request

6. Click Add from Price List under Delivery Item. The list of prices appear. This list is populated from the Price List workspace.
7. Choose HEAT Cloud Service Management 2014.1 subscription, then click Select.

Sample Request Offering - Select Price List Items

The item now appears in the request offering. This represents the base delivery item for this product and is charged to all users.

8. Click Design Request Form. This is where you define optional packages, such as Phone or Service Plan.
9. Click Edit next to Phone. The Configuration Editor appears.

Sample Request Offering - Item Configuration Editor for Phone

10. Open the Pick List section, then click Add. A new line appears.
11. In the first field, type Blackberry, then click Pick Price. The Select Price List window appears.
12. Choose Blackberry from the list (the one you just created), then click Select.
13. Click OK.

You have now added an item to the list, which appears to users in the Service Catalog.
