Deleting a Request Offering

After you create and publish a request offering, you can hide or delete it:

To hide a Request Offering—To remove a request offering from the Service Catalog, but NOT delete it from the system, save it in draft mode following the procedures described in Working with Request Offerings.

Delete a Request Offering

1. Log into the Service Desk Console as a service owner and open the Request Offerings workspace.
2. Find the request offering that you want to delete.
3. Click the request offering name link to go to the form.
You can only delete a request offering that you created.
4. Click the Publish Action tab, and select sesign mode.
5. Click Save.
6. Click Delete next to the request offering that you want to delete.
When you delete a request offering, the fulfillment workflow associated with it is also deleted. If a request is in the process of being fulfilled when you delete its associated request offering, it cannot be fulfilled through the workflow that was originally set up. To avoid this situation, it is recommended that prior to deleting a request offering, you keep it in design mode for a period of time that is sufficient to make sure that the fulfillment workflows for all pending requests are able to finish.
