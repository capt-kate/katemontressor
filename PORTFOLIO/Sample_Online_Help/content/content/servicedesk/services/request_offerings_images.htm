Request Offerings Default Icons

Listed are the default images (icons) used within request offerings. These images appear to users in the Service Catalog.

You can use any of these images when creating a request offering. If the image does not appear in the application, then right-click the image from this table in your browser and save to your desktop. You can then move the image into the appropriate folder for access. Add Images to the List Control for procedures on how to use these images.
