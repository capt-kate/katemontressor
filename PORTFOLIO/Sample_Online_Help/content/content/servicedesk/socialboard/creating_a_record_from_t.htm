Creating a Record from the Social Board

By default, HEAT provides the ability for you to create an incident, knowledge article, problem, release, or change from the Social Board. This capability can be extended to other objects.

The Insert Child Object quick action is used for creating the new business object record. These quick actions are created for the social feed business object from the Quick Actions Editor in the Configuration Console.

Create a Record

Create an Incident, Change, Problem, or Release Record from the Social Board
1. Within a Social Board message, click the settings drop-down list and choose Create Incident or Create Change. The message window appears.
2. Enter information into the fields.
Field Description
Service The name of the service.  The default is Service Desk.
Category The name of the category. The default is Service Desk.
Summary This field is auto-filled from the message. You can modify it as needed.
Description This field is auto-filled from the message. You can modify it to show a detailed explanation.
3. Click Share.

The incident or change record is created and the record number appears in the message.

Record is Created

  • The incident status is set to active and the priority is set to 5. The incident is assigned to the creator.
  • The change status is set to logged and the change type is set to minor.
  • The problem status is set to logged and the priority is set to 5.
  • The release status is set to draft and the priority is set to 3.
Create a Knowledge Article from the Social Board
1. Within a Social Board message, click the settings drop-down list and choose Create Knowledge Article. The message window appears.
2. Select the type of knowledge article:
  • Document
  • Error Message
  • Issue & Resolution
  • Patch
  • Q&A Reference
3. Enter information into the fields.

Document
Category Choose from the drop-down list.
Collection Choose from the drop-down list.
Title Enter a title for the document.
Description Enter the details of what is being documented.

Error Message
Category Choose from the drop-down list.
Collection Choose from the drop-down list.
Title Enter a title for the error message.
Description Enter an explanation of how the error message is caused.
Resolution Enter a description of how the issue was resolved.

Issue & Resolution
Category Choose from the drop-down list.
Collection Choose from the drop-down list.
Title Enter a title for the error message.
Description Enter an explanation of how the error message is caused.
Resolution Enter a description of how the issue was resolved.

Patch
Category Choose from the drop-down list.
Collection Choose from the drop-down list.
Patch Name Enter the name of the patch.
Installation Instruction Enter the installation details.

Q&A
Category Choose from the drop-down list.
Collection Choose from the drop-down list.
How do I?
How can I?		 Enter the question.
Answer Enter the answer to the question.

Q&A Reference
Category Choose from the drop-down list.
Collection Choose from the drop-down list.
Title Enter the question.
Description Enter the detailed answer.
4. Click Share.

The knowledge article is created in draft status.
