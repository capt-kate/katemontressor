Preventing Profile Modification

1. From the Configuration Console, click Configure > Users and Permissions > Roles and Permissions.
2. Click the role name to which you want to prevent changing the profile.
3. Choose the Object Permissions tab.
4. Select the Profile object.
5. Uncheck Edit.
6. Click Save from the toolbar.
