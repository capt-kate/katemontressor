Changing Locales

If your administrator has given you privileges, you can change the language that you see when using HEAT.

Do the following to change your locale:

1. From the header bar, hover over your name.  
2. Click the down arrow and choose Change Locale. A list of languages that your administrator has set up appears.

Available Languages

3. Select the language to use. Some languages might appear more than once. For example, English might appear as USA and Australia. Each version reflects cultural language features of the indicated country.

HEAT automatically refreshes with the changed locale.
