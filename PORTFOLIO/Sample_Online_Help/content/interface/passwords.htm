Passwords

Passwords are created when the administrator creates user records. Internal user passwords are stored in the user database, but external user passwords are not.

Changing Your Password

To change your password within HEAT, do the following:

1. From the top header bar, select your user name, then click Change Password.

Change Password

The Change Password dialog box appears.

2. Enter the old password and the new password in the appropriate boxes, then click Change.
3. Log out and log in again to use the new password.
If You Forget Your Password
This feature is only available for internal authorization and does not work for external logins.
1. If you forgot your password, on the main login window, click Forgot password?

Forgot Password

The Password Recovery window appears.

Password Recovery

2. Enter your user name or your email address.  The system sends you a temporary password.
3. Use the temporary password to log in, then change your password as described above.
