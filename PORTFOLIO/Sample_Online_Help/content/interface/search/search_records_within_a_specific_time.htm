Finding Records that Will Breach

This topic describes how to create a saved search that searches for records that will breech within a specific time using an expression. This example finds records that will breach in the next 7 days.

1. Within the Service Desk Console, open a workspace such as Incident.
2. Click Saved Searches to open the list of saved searches, then click New Search to create a new search. See Using Saved Searches for more information on how to create a saved search.
3. Enter the following parameters:

where    Incident Status Not in List Closed, Resolved

and      Escalation Watch.IncidentAssocResolutionEscWatch BreachDateTime Less than $(NextDays(7))

and      Escalation Watch.IncidentAssocResolutionEscWatch BreachDateTime Greater than $(CurrentDate())

Search the Next 7 Days

4. Preview to check your search results, then Save.
