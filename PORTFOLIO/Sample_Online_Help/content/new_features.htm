Analytic Metrics - Historical Data Analysis
  • Allows the creation and management of analytic historical snapshots.
  • Supports creation of historical snapshot from previous history.

An analytic snapshot lets you report on historical data from the Analytic Metric Dashboards. For example, an SDA manager could set up an analytic historical snapshot that reports on the active P1 incident for the day from which he can create a dashboard showing the trending of P1 incidents over time.
