The process to create and resolve incidents generally is:

1. Create an incident record. See Working with Incidents.
2. Assign the incident to an employee. See Assigning an Incident.
3. Investigate the incident. See Investigating an Incident.
4. Resolve the incident. See Resolving Incidents.
5. Close the incident. See Closing Incidents.
