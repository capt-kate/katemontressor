Error Messages

Exception: Exception: Object Delete: Role SelfService Has No Rights to Delete ServiceReqParamLink#

System Exception: Object Delete

This message occurs when the Self Service role has the ServiceReqParam and ServiceReqParamLink object permissions NOT set to Delete and the Self Service user attempts to delete a saved service request.

Self Service Permissions

To fix this:

1. Within the Configuration Console, click Roles and Permissions > Self Service > Object Permissions.
2. For the ServiceReqParam and ServiceReqParamLink objects, check Delete (the last checkbox).
3. Click Save.
Information: The Following Information Does not Satisfy Validation Constraints

WeekNumber of Monthly Schedule: Field value must be provided

WeekNumber of Monthly Schedule: field is empty, and empty value is not in the validation list

Schedule Entry Error

This error occurs if HEAT Release 2012.2 and HEAT Release 2012.3 users are running ScheduleEntry.MonthlySchedule and ScheduleEntry.YearlySchedule. See Schedule Entry for information on resolving this error.
