Supported Data Types

Four types of fields are supported by expressions:

  • Boolean: Also known as logical fields.Used for storing BooleanUses AND, OR, and NOT as operators between expressions to produce results based on whether the complete expression is true or false. Also called logical. values such as true or false.
  • Number: Used for storing numerical values, for both integer and real numbers.
  • Text: Used for storing textual (such as string) data, with support for Unicode characters.
  • DateTime: Used for date and time values.  You can optionally include time zone information.

Five  other field types are available in HEAT (see Using Fields), but there are no expression use cases for them.
