IsModuleEnabled

Checks to see if a module is enabled.

Syntax

IsModuleEnabled(value)

Enabled for

Search/Dashboard No
Business Rules No
Quick Actions Yes
Workflow Objects No
Reports No

Parameter

value The name of a module.  

Return Value

Boolean value. Returns true if it is enabled or false otherwise.
