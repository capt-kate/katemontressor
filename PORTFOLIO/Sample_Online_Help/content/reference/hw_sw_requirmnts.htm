Software

Because HEAT is browser-based, it can run on operating systems that support browsers, including Windows, Linux, Unix, and Macintosh. We recommend using Chrome or Firefox due to increased performance of the Java Script engine; however, other browsers are compatible, such as Internet Explorer and Safari. Note that Internet Explorer Release 8 is particularly slow.

HEAT is also compatible with mobile devices and many mobile operating systems, including Android, iOS, and Windows. We highly recommended having a minimum of 1024x768 resolution, which makes tablets a better choice for use as a mobile browser than mobile phones.

  • All browsers must support Adobe Flash, which is required to use dashboard charting, pivoting, copy and paste control, and Service Catalog attachment control.
  • If your HEAT system includes integrated components (such as FrontRange Voice), you must use the Windows operating system.

We recommend the following workstation specifications:

Web browser  
HEAT Cloud is a web-based application. HEAT Cloud requires the web browser versions listed below. Using versions of web browsers that are not listed in this table may result in improper functioning of the application.

  • All security updates are recommended.
  • The browser text size must be set to Normal.
  • The screen resolution or space allocated for the browser should be 1280 X 1024 (or higher) when accessing the Configuration Console.
Google Chrome
  • Only the latest production version is supported. This does not include Beta versions of Chrome.
Mozilla Firefox
  • Only the latest production version is supported.
Microsoft Internet Explorer 8.0 and above. Version 9.0 is recommended.

Set the following options to enable on the Tools > Internet Options > Security > Custom level page:

  • Run ActiveX controls and plug-ins
  • File download
  • Scripting > Active scripting

To enable Adobe Flash, go to Tools > Manage add-ons and Enable Adobe Shockwave Flash Object.

Apply all Microsoft hot fixes.

HEAT also works with the following browsers:
Safari
  • Only the latest production version is supported. Safari is not supported on iOS.
Additional requirements

Windows 7 or newer
  • For optimal performance. Older operating systems may function, but at a reduced performance level; use at your own risk.
Microsoft Office 2003, 2007, or 2010
  • Required for the Export to Excel feature.

Adobe Flash
  • Required in the browser to support charts. Make sure to install the Adobe Flash plugin to make charts work.
  • For operating system and browser compatibility with the Adobe Flash player, see http://www.adobe.com/software/flash/about/

Display Settings

We recommend the following display settings for workstations:
 
  • Screen area: 1280 x 1024 or higher
  • Font size: Small fonts or normal size (96 dpi)
Networks  
 
  • HEAT requires a high-speed Internet connection with a latency of 110 MS or less for good performance.
  • Port 443 (HTTPS - SSL) must be open for connection.

Third-Party Software Components

The HEAT installation package provides the following required third-party software components:

  • Microsoft .NET Framework Releases 3.5 and 4.5
  • Microsoft SQL Server 2012 Management Object (SMO) (redistributable)
  • Microsoft Access Database Engine 2007 and 2010 (redistributable)
  • Windows Server Roles and Features

If the installer does not detect these components on the host system, they are installed automatically.

Using a Virtual Machine

You can also deploy HEAT as a virtual image. If you use a virtual machine, note the following:

  • When you copy the virtual image into a new machine, update the MAC address through the HEAT License Server.
  • Preserve the unique identifier.
