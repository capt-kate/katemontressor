Problem Quick Actions

Action menu option

Action taken

Resolve via Workaround

Updates all of the incidents linked to the problem and moves them to the status of resolved. Copies the workaround information from the problem to the Resolution Detail tab of the incidents.

Resolve via Resolution

Updates all of the incidents linked to the problem and moves them to the status of resolved. Copies the resolution information from the problem to the Resolution Detail tab of the incidents.

Request for Change

Creates a new change from the problem record.Bug pending: If implemented, will link all CIs from change to the problem.

Calculate Cost

Updates the Cost field of the problem by rolling up the cost of all completed tasks that are associated with the problem.

New Knowledge Article

Creates a new knowledge article, based on the contents of the current problem.

Close Related Incidents

Closes all incidents related to the current problem.

Problem Status to Closed

Updates the status of the problem to closed.
