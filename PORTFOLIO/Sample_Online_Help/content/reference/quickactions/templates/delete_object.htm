Delete Object Quick Action

Deletes the current context object. This quick action has no specific parameters, other than Name and Description. Use this quick action within a Search.

Since this quick action deletes the current context object, use care when including it in a Composite quick action.

Use extreme caution if using this in a Composite action as it could potentially delete the wrong value.
