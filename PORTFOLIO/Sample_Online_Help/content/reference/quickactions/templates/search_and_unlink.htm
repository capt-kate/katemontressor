Search and Unlink Quick Action

Searches for and unlinks a business object record from another record. For example, you can unlink a CI record from an incident record.

Search and Unlink Quick Action

Parameter Description
Relationship The relationship between the current object and the object to which you are linking.
Find where Where to search.
Operator A comparison operator.
String A string to use for the comparison.
  • Click Add to add a search filter to the list.
  • Click Delete to remove a filter from the list.
