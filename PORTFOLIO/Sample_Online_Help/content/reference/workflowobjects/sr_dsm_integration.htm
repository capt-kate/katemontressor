SR DSM Integration Workflow Block

SR DSM Integration Workflow Block

Exit Ports

ok

Continue to the next block.
failed

General exit port for a failed integration action. A logical next step is to trigger a notification.

Properties

Title Enter a unique name for the object.
DSM Request Type Field

Choose the request type from the drop-down list. Other fields appear based on your selection.

  • Provision computer
  • Reinstall computer
  • Install software
  • Uninstall software
Computers Choose from the drop-down list.
Hardware Profile

Appears when you choose Provision Computer from the DSM request type field.

Choose an object from the drop-down list.
Software Provisioning Profile

Appears when you choose Provision Computer from the DSM request type field.

Choose an object from the drop-down list.
Software List

Appears when you choose Provision Computer from the DSM request type field.

Choose the software from the drop-down list.

Click Add to add a field and choose more software. Repeat as needed.

Click Delete next to the field that you want to remove from the list.
Execute Immediately? Choose the execution type from the drop-down list.
Recalculation Installation Order?

Appears when you choose Reinstall Computer from the DSM request type field.

Choose an installation order from the drop-down list.
FRS Software

Appears when you choose Install Software from the DSM request type field.

Select an object from the drop-down list.
Package or Set

Appears when you choose Install Software from the DSM request type field.

Select an object from the drop-down list.
Software

Appears when you choose Uninstall Software from the DSM request type field.

Select an object from the drop-down list.
  • Click Save to commit your settings.
