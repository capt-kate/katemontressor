Updating a Request

You can update a service request by adding an attachment (such as a screen image, a document, and so on) or by adding additional information about the service request in a note.

Update with an Attachment
1. Open the service request as described in View a Request in the My Items Tab.
2. Click the Current Status tab.
3. In the Add an Attachment field, click Browse. Navigate to the file location. Select the file to attach it.

After you add an attachment:

  • A Service Desk Analyst can view the attachment in the Attachment tab of the service request record.
  • You can view your attachment, along with other request details, in the My Items tab.
Update with a Note
1. Open the service request as described in View a Request in the My Items Tab.
2. Click the Current Status tab.
3. In the Add a New Note field, enter additional information for the service request.
4. Click Add Note.

After you add a note:

  • A Service Desk Analyst can view it in the Activity History tab of the service request record.
  • You can view your note, along with other incident details, in the My Items tab.
