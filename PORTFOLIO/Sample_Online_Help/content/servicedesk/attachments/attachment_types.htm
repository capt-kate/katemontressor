Attachment Types

The default attachment types are:

.avi .bmp .cer .csv
.doc .docx .eml .flv
.gif .html .jpg .mht
.mhtml .msg .pdf .pfx
.png .pps .ppt .pptx
.rar .rtf .tif. .txt
.wmv .xls .xlsx .xml
.xps .zip    

 

Adding an Attachment Type for Your Organization

Create attachment types not listed.

1. Within the Configuration Console main page from the Settings menu, go to Configure > Security Controls > Allowed File Extensions.

The Allowed File Extensions page appears. The current file extensions are listed.

2. Click Add New. The Add File Mask window appears.
3. Enter the file type including the dot (.), then click Save.

The new type appears in the list.

Deleting an Attachment Type

Within the Allowed File Extensions workspace, click Delete next to the file type to remove, then click Yes to confirm the deletion.

Delete an Attachment File Type

The type no longer appears in the list.

Deleting a file type from the list does not remove attached files from the database.
