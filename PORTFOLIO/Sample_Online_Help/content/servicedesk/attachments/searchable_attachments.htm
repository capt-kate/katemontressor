Searchable Attachments

You can attach a document to a knowledge article that can be found during a search. This document is enabled for a full text search. File formats supported are txt, doc, docx, xls, xlsx, and pdf. Other formats can be attached here, but will not be searchable.

Ensure that your Microsoft SQL server is enabled to search PDF files, if you plan to use this feature.

You can only attach one searchable document to a knowledge article at a time. If you attempt to upload another document, the original document is replaced by the new document.

Attaching a Searchable Document
1. Within the knowledge article, under the Details tab, select Upload next to Add Searchable Document. The file navigation window opens.
2. Navigate to a document and click Open. The file is uploaded and the truncated file name appears.
3. Click Save from the toolbar.
Viewing a Searchable Document

You can view a searchable document within a knowledge article.

1. Within the knowledge article, under the Details tab, click the document title. The Opening window appears.
2. Select an application (such as Microsoft Word) and click OK. The file is downloaded to your local drive and opened for reading.
Removing a Searchable Document

Within the knowledge article, under the Details tab, click Delete next to Add Searchable Document.
