Budget Items

You can designate specific items to be included in a budget. Budget Items are added to the budget within the Sub-Budget Plan, which is then included in the Master Budget Plan.

Create a Budget Item
1. Within the Sub-Budget Plan workspace, select the Budget Item tab. The list of budget items appears, if any.
2. Click New ITFM Budget Item. The New ITFMInformation Technology Financial Management. Part of the Financial Management capabilities of HEAT. Budget Item window appears.
3. Enter information into the fields.
--fields--
Field Description
Budget Item # This unique number is automatically generated by the system.
Name Enter a unique, descriptive name for the budget item.
Description Enter a description of the budget item.
Price List Item Enter a price list item from the list.
Cost Driver This number is automatically entered by the system based on the chosen Price List Item.
Unit Cost This number is automatically entered by the system based on the chosen Price List Item.
Usage Enter the number of units you plan to purchase or the number of hours used.
Budget This amount is automatically calculated by the system based on the unit cost multiplied by the usage number. When this item is calculated, the total amount in the Sub-Budget and Master Budget Plans are updated.

Budget Start Date

Budget End Date

 If the sub-budget is created from the master budget, these dates are automatically entered by the system. Else you can enter the dates you want for your sub-budget.

Capital Expense

Direct Cost

Fixed Cost

 These items are automatically checked based on the chosen Price List Item.
4. Click Save. The window closes and the item appears on the list.
