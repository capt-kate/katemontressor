Deploying the MDI Client from a Mobile Device

End users must install the client application on their mobile device by downloading it from Google Play for Android devices, or from the Apple Apps store for iOS devices.

Configure the MDI Client Application

After you install the client application on the device, open the application and do the following:

1. Enter the user name in the User Name field.
2. Check Company Owned, if appropriate.
3. Enter the server URL that was sent by the administrator.
4. Enter the customer ID that was sent by the administrator.

Enroll a Mobile Device

To enroll the mobile device with MDI, do the following:

1. Tap Enroll.
2. Optionally, to secure the enrollment process, an administrator may require an enrollment password. This password is used only once, during the initial enrollment of a device. If required, enter the password sent by the administrator.
3. The system prompts you for confirmation to activate the service. You must agree, and then install the certificate sent by the MDI service.
