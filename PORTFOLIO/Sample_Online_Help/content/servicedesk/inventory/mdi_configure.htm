Configuring Mobile Device Inventory (MDI) Settings

After you install HEAT Discovery , you can configure the Mobile Device Inventory (MDI) settings.

MDI Tab

Edit the following parameters on the MDI tab by opening a tenant record in the HEAT Configuration Database (ConfigDB):

  • MDI_Explanation
  • MDI_LogoURL
  • MDI_Password

.

See the Configuration Database Guide for HEAT Service Management for more information about these parameters.
