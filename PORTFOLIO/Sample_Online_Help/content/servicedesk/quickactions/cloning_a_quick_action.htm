Cloning a Quick Action

You can create an exact copy of a quick actionA user-defined operation that performs one or more actions with a single click of the mouse., then easily modify any parameter.

1. Within the Quick Actions CenterAllows you to create and edit quick actions associated with the workspace or object context., select the Your Actions tab to view a list of quick actions.
2. Select a quick action from the list.

Cloning a Quick Action

3. Click Clone from the Button area to activate the configuration interface.
4. Click the quick action name and enter a new name. The application only runs the first quick action encountered if there are duplicate names.
5. Edit any fields as needed, then click Save.

The quick action now appears under the Your Actions list.
