Using the Run Program Quick Action

This procedure describes how to create a quick actionA user-defined operation that performs one or more actions with a single click of the mouse. that executes a file or script on a remote server.

1. Within the Quick Actions CenterAllows you to create and edit quick actions associated with the workspace or object context., under the Create New Actions tab, choose Run Program. The Run Program Execution page appears.
2. Enter the name of the quick action, then enter information into the fields. See Run Program Quick Action quick action parameters.

Run Program Quick Action

3. Click Save from the toolbar.
