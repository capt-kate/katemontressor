Using the Run for Child Quick Action

Use the Run for Child quick actionA user-defined operation that performs one or more actions with a single click of the mouse. to perform an action on a child object of a business object. For example, you might want to notify a release milestone owner when a release milestone is in active status. In this example, release is the object and milestone is the child.

1. Open the Quick Actions CenterAllows you to create and edit quick actions associated with the workspace or object context. for the release business object.
2. From the Create New Actions tab, choose Run for Child.
3. Enter information into the fields.
--fields--
Field Description
Name A name for this quick action.
Category The category. Select from the drop-down list.
Publish to The roles that need access to this quick action.
Relationship Select ReleaseAssociateMilestone.
Action to Run Select Milestone Owner Active Milestone Notification.
4. Click Save.
