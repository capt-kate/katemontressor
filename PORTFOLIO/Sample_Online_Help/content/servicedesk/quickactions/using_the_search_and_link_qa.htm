Using the Search and Link Quick Action

You can use the Search and Link quick actionA user-defined operation that performs one or more actions with a single click of the mouse. to perform an action on a child object of another child object that is associated to a business object. For example, you might want to search for all configuration items associated to a change that is associated to a release in order to directly link all the configuration items from the change to the release.

1. Open the Quick Actions CenterAllows you to create and edit quick actions associated with the workspace or object context. for the release business object.
2. Click the Create New Actions tab, and then click Search and Link.
3. Enter information into the fields.
--fields--
Field Description

Name

 A name for this quick action.
Category The category. Select from the drop-down list.
Publish to The roles that need access to this quick action.
Save object Saves the action automatically.
Relationship Select ReleaseProjectAssocCI.
Find CI where

Select CI from the first drop-down menu, then enter [Change#.][ReleaseProject#.]RecId. Then select Begin with and enter $(RecId).

4. Click Save.
