Releases

Release encompasses planning, design, build, configuration and testing of hardware and software releases to create a comprehensive release strategy. Release Managers and administrators can view releases.

View Release Records
  • Log into the Service Desk Console as a Release Manager and open the Release workspace. A list of all the releases, arranged from the highest release ID appears.
  • Double-click an item from the list to view details.
