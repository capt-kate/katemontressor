Scheduling a Report

You can schedule a report to run at specific times.

Within the Report workspace in the Schedule Information section, choose from the following:

Option Description
No Schedule Check to NOT run the report on a schedule.
Run once on Enter a date or select a date from the calendar, then choose a time from the drop-down list.

Daily at

 Check to automatically run this report every day, then choose a time from the drop-down list.

Weekly on

 Check to automatically run this report each week, choose the days you want it to run, then choose a time from the drop-down list.

Monthly (by Week) on

Check to automatically run this report on a specific week each month, then choose the months you want from the drop-down list. For example, you could run this report every month or every other month.

Choose the week from the drop-down list:

  • First
  • Second
  • Third
  • Fourth
  • Last

Choose the day of the week to run the report, and then choose the time of day from the drop-down lists.
Monthly (by Day) on

Check to automatically run this report on a specific day each month, then choose the months you want from the drop-down list. For example, you could run this report every month or every other month.

Type the day number you want to run the report. For example, if you want to run the report on the 4th of every month, type 4. To run the report on the 5th, 10th and 15th of the month, type 5, 10, 15, separated by commas. To run the report within a date range, say from the 5th to the 10th, type 5-10.

Start

End

 All scheduled reports can be designated to start and end on specific dates. Select a start and end date from the calendar, or enter your dates into the fields.

After the scheduled report runs, it is available for download from the Report Center (only if the download option is selected) or emailed (only if the option is selected). The Schedule Status indicates if the report was successfully run and the time that it ran.
