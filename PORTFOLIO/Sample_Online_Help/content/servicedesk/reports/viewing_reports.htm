Viewing Reports

To make a report available to a certain user role, the administrator must do the following first:

  • Make the Report workspace available to that user role.
  • Publish the report to that user role.

You can only view reports published to your role. Each role has different reports available.

 

There are three ways to view reports:

  • View Reports from the Report Workspace
  • View Reports from a Dashboard
  • View Reports from the Quick Actions Menu
View Reports from the Report Workspace

You can only see reports available to your user role.

Follow these steps to view reports from the Report workspace:

1. Log in to the Service Desk Console.
2. Open the Report workspace. A list of reports published to your role appears.
3. Click the report name to generate a report.
4. Some reports allow you to set parameters. Set your parameters in the header area of the report, then click View Report.
View Reports from a Dashboard

To view a report from a dashboard, you must first add the report to a dashboard. Follow these steps to add a list of reports to your dashboard and then to view the list:

1. From your Home workspace, click My Dashboards. A list of dashboards appears.
2. Select Edit next to the dashboard to which you want to add the report list as a part. The dashboard interface appears.
3. Expand the Special Parts tab from the accordion menu.
4. Drag and drop a report list from the accordion menu to the dashboard area. The report list appears on the dashboard.
5. Click Home from the toolbar to return to the Home page. The dashboard part appears, listing the reports available to your role.
6. From your Home workspace, select the report that you want from the report list. The report opens in a new window.
View Reports from the Quick Actions Menu
1. From the Service Desk Console, open a workspace, such as Incident or Change.
2. Select the records that you want to view (you can select one or more records).
3. From the Action Menu, choose Print > Print <business_object_name>. The report appears in a separate window.
