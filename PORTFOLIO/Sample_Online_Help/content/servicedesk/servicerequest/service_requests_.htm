Service Requests

An end-user request that is not a failure (incident) is a request for service (service request). Service requests are usually initiated within the Self Service dashboard.

In HEAT, Service Desk Analysts might create service requests for a requester or provide details on an existing request.

Service Catalog request items are created by service owners.

See Direct Links to Service Requests for examples on how to access service requests from a URL.

View Service Request Records
  • From the Service Desk Console, open the Service Request workspace. A list of service requests appears.
  • Double-click a service request record to see details.
