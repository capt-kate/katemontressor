Exporting a Request Offering

You can export a request offering template to your local drive.

1. Within the Request Offering workspace, click Save Template on a Disk next to the request offering to copy.

Request Offering Workspace

The export window appears.

Export Window

2. Check Save File and click OK.
3. The template is saved to your local drive, probably into your downloads folder.
