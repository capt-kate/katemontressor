Outage Log

The Outage Log captures information on a service outage and is part of the Availability Management process. Each time you have an outage, record it here. The Availability Report deducts this time from the availability.

Please note: as of HEAT version 2014.2.0, this feature is not fully functional.

Create an Outage Log
1. From the Service or CI Service workspace, select the Outage Log tab. (See Creating a Service.) The list of logs come up, if any.
2. Click New Outage Log. A blank form comes up.
3. Enter information into the fields.
--fields--
Fields Description
Category Choose a category from the drop-down list, or type in a new category.
Started On The current time appears by default. Click the calendar to choose a different date and time.
End On The current time appears by default. Click the calendar to choose a different date and time.
Duration Optional. Enter the number of seconds you want to capture.

Then click Save. The window closes.
