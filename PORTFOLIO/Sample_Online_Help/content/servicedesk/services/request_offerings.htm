Request Offerings

Request offerings are items that describe the item, assistance, or action available to customers from the service catalog. Request offerings are normally placed in logical groups of service offerings. Both service offerings and their request offerings are available to Self Service users when the status of the offerings is set to published and if end users have been assigned a corresponding Service Manager user role. Only users who have been assigned a user role associated with a catalog group that contains catalog items can use the Self-Service portal to access the service catalog.

View Request Offerings Records

Only service owners and administrators can view the request offerings list, unless role access has been modified by the administrator.

  • Within the Service Desk Console, open the Request Offerings workspace. A list of offering records appears.
  • Double-click a record to view details.
