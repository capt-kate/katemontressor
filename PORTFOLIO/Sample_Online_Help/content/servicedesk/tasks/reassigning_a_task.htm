Reassigning a Task

If you are unable to work on a task or if the task is escalated, you might have to reassign the task.

1. From the task toolbar, click Reassign.
2. To assign another team to work on the task, select a team and click OK. The task status changes to logged.
To assign an owner to the task, open the task record and select an individual from the Owner menu.

HEAT notifies the task assignee by email of the task assignment. If an owner is not specified, the task remains in the logged state until it is assigned to an owner.
