Task Catalog

The task catalogLists external tasks used throughout HEAT. lists external tasks used throughout the application. Tasks are created within an underpinning contractAn Underpinning Contract is a Contract between an IT Service Provider and a Third Party. The Third Party provides goods or Services that support delivery of an IT Service to a Customer. The Underpinning Contract defines targets and responsibilities that the Service Provider will use to determine Service Level Targets and Packages. or service level agreementAn agreement between the service provider and the customer documenting the agreed upon service levels for the service., and appear here automatically.

Creating a Task Catalog Item

You can only add external tasks to the task catalog.

1. Within the Underpinning Contracts workspace, within a record from the Task Catalog Item tab, click New Task Catalog Item. A blank form appears.
2. Enter information into the fields.
--fields--
Field Description
Name A unique name for the external task.
Status The task catalog status. Choose from the drop-down list. See Task Catalog Status.
Task Subject The task title. This appears in the subject line of subsequent emails.
Task Details A description of the task. This appears in the body of subsequent emails
Task Team The task team. This field is automatically generated by the system according to the parent record.
Task Owner The task owner. Select from the drop-down list or enter a name.
Vendor The vendor name.
Vendor Contact Name This field is generated based on the Vendor field selection.
Vendor Contact Email This field is generated based on the Vendor field selection.
Vendor Contact Phone This field is generated based on the Vendor field selection.
Job Item The job item. Choose from the drop-down list. This list is based on the price list Item associated with the vendor. See Price List and Using External Tasks.
Job Unit Price The unit price.  Based on the selected job item.
3. Click Save. The window closes and the task appears on the list.
